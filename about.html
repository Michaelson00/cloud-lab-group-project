The Project

Project Details

Our Cloud Computing Practical Lab 1 covers two major areas: virtualization setup and static website deployment.

Objective

Our Goal

To gain hands-on experience with cloud computing fundamentals — specifically virtualization using Oracle VM VirtualBox and web deployment using GitHub Pages.

Tools

Technologies Used

Oracle VM VirtualBox, GitHub, GitHub Pages, HTML5, CSS3, JavaScript — the foundational tools of cloud and web development.

Process

Our Approach

We divided responsibilities clearly: a leader to coordinate, a repository manager to handle GitHub, and dedicated members for design and content creation.
